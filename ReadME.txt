说明：
本工具是用于浅浅幻想之旅录入服装xml数据用。


使用方法：
0.先将文件编码转为utf-8
1.打开软件：xmlTool4浅浅.exe
2.将xml文件拖入指定范围
3.选择设置套装/部件，添加内容
4.在添加完内容后记得点击应用【包括在切换设置模式时】
5.点击导出，添加了内容的新文件将会出现在原文件的同一路径下的'_增加后.xml'

</清新>

</保暖>
